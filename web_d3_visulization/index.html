Bike sharing systems are widely used in many cities around the world, which are cost-efficient and environment-friendly for both users and cities.

One appealing feature is that it allows people to borrow a bike from one station and return it at another station.

Although this advantage provides incredibly convenience for the riders, it poses the rebalancing problem for the system.

For assisting the rebalancing issue, our system - VizBike@Pittsburgh can be used to visualize the bike share data provided by Healthy Ride in Pittsburgh from 2015 Quarter 3 to 2016 Quarter 2.
